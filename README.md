## Commit Rule

커밋의 타입은 아래와 같이 4가지로 구분

- New : 새로운 문서 작성
- Update : 문서 내용 수정
- Fix : 문서 오타 수정
- Delete : 문서 삭제

```
[New] Study Title.md

[Update] Study Title.md

[Fix] Study Title.md

[Delete] Study Title.md
```
